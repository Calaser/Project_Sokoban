# Project_Sokoban
一個畫面由文字組成的倉庫番小遊戲。

[**>> 點此前往網站 <<**](https://calaser.github.io/Project_Sokoban)

<img src="https://i.imgur.com/3G6ANM5.png">

## 網站內容
收錄了由純文字構成畫面的倉庫番小遊戲，<br />
目前共有10個關卡可供挑戰。

## 網站功能
有三套按鍵可以操控上下左右
+ WASD
+ 十字鍵
+ 數字鍵盤 2/4/6/8

可以重新開始關卡，<br />
並且可以選擇上一關、下一關、從選單中選擇任意關卡等，<br />
以上功能除了使用滑鼠操作以外也有對應的快捷鍵可以按。

另外會紀錄已經通關的關卡。
